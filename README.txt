Jirc is a Java translation of substantial parts of Lirc 0.9.0.  It consists of a
library and the executable program Lirc2XML which converts one or more
Lirc configuration files to Girr XML format.

Authors: Christoph Bartelmus, Bengt Martensson, and others.
